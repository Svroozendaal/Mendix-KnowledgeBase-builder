# STATUS
## Purpose

Provide a quick, structured snapshot of current work status.

## Steps

1. Read `.agents/AGENTS.md`.
2. Ask clarifying questions first if scope is unclear.
3. Summarise:
   - Current goal
   - Progress to date
   - Open blockers
   - Suggested next action

## Output Template

```markdown
## STATUS
CURRENT_GOAL: [...]
PROGRESS: [...]
BLOCKERS: [...]
NEXT_ACTION: [...]
```
