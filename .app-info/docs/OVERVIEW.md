# docs/
## Produced Documentation

This folder stores application-level documentation produced during development.

## Documents in this folder

- `PRODUCT_OVERVIEW_CHATBOT.md`: single-document product overview designed to be pasted into a chatbot for product questions.
- `PARSER_ARCHITECTURE.md`: parser architecture and design decisions.
- `MODEL_OVERVIEW_EXPORT_CONTRACT.md`: v2.0 export format contract.
- `MODEL_DUMP_PARSER_GAPS_2026-02-27.md`: historical parser gap findings from dump inspection.
- `KNOWLEDGEBASE_CREATOR_ARTIFACT.md`: portable artifact product direction and ownership.
- `GAPSMITH_TODO_TEMPLATE.md`: standard backlog template for parser vs AI interpretation gaps.

## What belongs here

- Cross-cutting architecture and behaviour summaries.
- Single-file product overviews for onboarding and chatbot context.
- Parser gap analyses and coverage reports.
- Export format contracts and specifications.
- Product-direction docs for portable KB creation packaging.
