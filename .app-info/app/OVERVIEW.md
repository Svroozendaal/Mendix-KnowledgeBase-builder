# app/
## Application Overview Folder

This folder contains the identity, vision, and high-level specification of the application.

## Contents

| File | Purpose |
|---|---|
| `PRODUCT_PLAN.md` | Application vision, goals, non-goals, and delivery model |

## When to Update

- When the product vision changes.
- When significant non-goals are added or removed.
- When the technology stack changes fundamentally.

## Notes

This folder is the first place an AI assistant should read to understand what the application is and what it is trying to achieve.
