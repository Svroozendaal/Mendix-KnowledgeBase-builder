# development/commands/
## App-Specific Commands

This folder contains command guidance specific to this application. These commands extend the generic commands in `.agents/commands/`.

## Contents

Add app-specific command files here as needed. Each command file should explain:
1. When to use the command.
2. What steps to follow.
3. What output or result is expected.

## Notes

Generic commands (document, implement, review, status) live in `.agents/commands/`.
App-specific commands here take precedence when there is a conflict.
