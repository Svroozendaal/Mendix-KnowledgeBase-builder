# development/
## Development Resources Folder

This folder contains all development resources specific to this application: prompts that drive development phases, and app-specific command guidance.

## Contents

| Subfolder | Purpose |
|---|---|
| `prompts/` | Phase prompts and development playbooks for this application |
| `commands/` | App-specific command guidance that extends `.agents/commands/` |

## When to Update

- When a new development phase is added.
- When an app-specific command or workflow is defined.
