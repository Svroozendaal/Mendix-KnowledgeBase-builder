# PHASE_5_REVIEW
## Goal

Perform final quality review and produce a release decision.

## Entry Criteria

1. Implementation and testing outputs are available.
2. Clarifying workflow questions have been asked as needed.

## Required First Action

1. Read `.agents/AGENTS.md`.
2. Ask workflow questions first.
3. Ask which skills should be used and suggest relevant defaults.

## Tasks

1. Review implementation and test evidence.
2. Classify findings by severity.
3. Approve or request changes with exact file references.
4. Write verdict to `SESSION_STATE.md`.

## Exit Criteria

1. `MUST FIX` status is explicit.
2. Review verdict is recorded.
3. Next action is clear.
