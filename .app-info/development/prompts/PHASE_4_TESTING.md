# PHASE_4_TESTING
## Goal

Confirm functionality, robustness, and regression safety.

## Entry Criteria

1. Implementation hand-off is available.
2. Clarifying workflow questions have been asked as needed.

## Required First Action

1. Read `.agents/AGENTS.md`.
2. Ask workflow questions first.
3. Ask which skills should be used and suggest relevant defaults.

## Tasks

1. Define expected behaviour and edge conditions.
2. Execute build/static/functional checks.
3. Record failures with reproducible steps in `REVIEW_NOTES.md`.
4. Issue PASS/FAIL verdict and hand off to Reviewer.

## Exit Criteria

1. Automated testing verdict is recorded.
2. Findings are logged with reproduction notes.
3. Reviewer hand-off is complete.
